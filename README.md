# Linux Bash Toolkit

This repository contains my hands-on practice with basic Linux shell commands and simple Bash automation.

## Topics covered

- File and directory navigation
- File creation, copying, moving, and deletion
- Viewing file contents from the terminal
- Absolute vs relative paths
- Simple Bash scripting for folder setup
